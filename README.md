# Advent of Code 2025

To run Haskell scripts, use `stack runghc <file>`. For example, to run day 1 part 1, run `stack runghc day01/part1.hs`.
To run day 10 part 2, run:
```
pip install pyscipopt
```
to install the linear programming library, then run the script with the `python` command.
